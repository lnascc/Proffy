👨‍🏫 Sobre
O Proffy é uma aplicação Web e Mobile com o objetivo de criar uma interação entre professores de diversas matérias e alunos. É possível cadastrar aulas e entrar em contato com o professor desejado.

Essa aplicação foi desenvolvida durante a Next Level Week #2, projeto da Rocketseat.
